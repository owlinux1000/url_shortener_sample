# Sample URL Shortener

A toy application to learn how to create service of URL shortening.

## 1. Setup backend

```
$ redis-server
$ ruby main.rb
```

## 2. Set up front

```
$ git clone https://github.com/owlinux1000/url_shortener_sample_front
$ cd url_shortener_sample_front
$ npm run dev
```
